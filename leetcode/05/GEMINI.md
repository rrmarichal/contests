# LeetCode Challenge: Longest Palindromic Substring

This directory contains a solution to the LeetCode problem "Longest Palindromic Substring".

## Problem Description

Given a string `s`, the task is to find the longest palindromic substring in `s`.

For example:
- If `s = "babad"`, the output could be `"bab"` or `"aba"`.
- If `s = "cbbd"`, the output is `"bb"`.

Constraints:
- `1 <= s.length <= 1000`
- `s` consists of only digits and English letters.

## Files

- `description.md`: Contains the problem description from LeetCode.
- `solution.go`: Contains the Go implementation of the solution.

## Running the Solution

To run the solution, you can execute the `main` function in `solution.go`. You can build and run the file using the following command:

```sh
go run solution.go
```
